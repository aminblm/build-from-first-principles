# Void to System: The First-Principles Ledger

> *"What I cannot create, I do not understand."* — Richard Feynman

Welcome to the definitive index of production-grade infrastructure, compiled entirely from the ground up. This repository strips away modern dependencies, forcing you to look directly into the raw logic of the systems you rely on daily. No packages. No frameworks. Just raw byte streams, sockets, and algorithms.

---

## The Core Mandates

* **Absolute Zero-Dependency:** Every project must compile or execute using only native standard runtime capabilities. No external imports allowed.
* **Protocol-Level Accuracy:** We do not mock dependencies; we implement raw RFC specifications (HTTP/1.1, TCP, Git wire protocol).
* **Architectural Visibility:** Code structures favor explicit data pipelines and clear component isolation over magical abstractions.
